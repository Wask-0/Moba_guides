# MOBA Guides Project 🎮

Учебный проект по разработке веб-приложения на фреймворке Bottle (Python). 
Сайт представляет собой сборник кратких гайдов по играм Dota 2 и Deadlock.

## Особенности
- **Стиль:** Темная киберспортивная тема с адаптивной версткой.
- **Движок:** Python + Bottle.
- **Шаблонизатор:** SimpleTemplate (TPL).

## Команда разработки (Authors)
* **wask0** — Team Lead, разработка раздела Dota 2, системная архитектура.
* **lasarc** — Developer, разработка раздела Deadlock, дизайн страницы авторов, маршрутизация.

## Как запустить
1. Установите зависимости: `pip install bottle`
2. Запустите файл `app.py`.
3. Откройте в браузере: `http://localhost:5555`

## Технологии
- Python 3.x
- Bottle Framework
- Bootstrap (для оформления карточек и сеток)
- Git/GitHub
